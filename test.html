1. Якою комбінацією клавіш можна швидко створити базову структуру HTML5-документа за допомогою Emmet?

а) html:5 + Tab
б) ! + Tab
в) doc + Tab
г) template + Tab

2. Яка команда на панелі команд (Ctrl+Shift+P) дозволяє налаштувати колірну тему VS Code?

а) Settings: Open Settings
б) Preferences: Color Theme
в) Theme: Change Color Theme
г) View: Appearance

3. Що робить комбінація клавіш Ctrl+` (зворотний апостроф) у VS Code?

а) Відкриває термінал
б) Закоментувати виділений код
в) Перемикає між відкритими файлами
г) Форматує документ

4. Який синтаксис Emmet створить навігацію із трьома пунктами списку?

а) nav>ul>li*3
б) nav>li*3
в) navigation>list*3
г) menu>items*3

5. Яке розширення рекомендується для Live Preview HTML-сторінок?

а) Live Server
б) HTML Preview
в) Browser Preview
г) Всі перелічені

6. Якою комбінацією клавіш можна швидко дублювати поточний рядок VS Code?

а) Ctrl + D
б) Alt + Shift + ↓
в) Ctrl+C, потім Ctrl+V
г) Усі варіанти вірні

7. Що дозволяє робити розширення "Auto Rename Tag"?

а) Автоматично перейменовувати парні HTML-теги
б) Автоматично закривати відкриті теги
в) Автоматично форматувати код
г) Автоматично додавати атрибути тегам

8. Який Emmet-абревіатурою можна створити div із класом "container"?

а) .container
б) div.container
в) container
г) Будь-який із перелічених

9. Що робить комбінація клавіш Ctrl+/?

а) Закоментувати/розкоментувати рядок
б) Розділити редактор на дві панелі
в) Пошук за файлом
г) Швидкий перехід до рядка

10. Яка функція VS Code дозволяє працювати з системою контролю версій Git без терміналу?

а) Source Control (Ctrl+Shift+G)
б) Git Integration
в) Version Manager
г) Code History
